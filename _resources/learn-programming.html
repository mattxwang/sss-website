Overview

Want to learn programming? Awesome! Here are a few resources to help you on your way to becoming a pro-grammer! We have resources to help beginner coders learn exactly what coding is, and how they can use it to help them out!

Want to add a resource? Awesome! Just make a pull request on our repo to add a resource.

Preface

Coding can look super scary, and it definitely can be! Simply put, programming is becoming a professional problem solver: whether it's solving a math question, figuring out how to best convey a message, or making your wildest dreams your reality! All you need to do is explain what you want to do, to your computer.

There are two large elements to any sort of programming: logic and syntax. Logic is the thought process behind your code, or a set of steps the computer will follow to solve your problem. Syntax is the format this logic is placed in, very similar to grammar in any language.

General Tools

Here are a few essential tools, no matter what language you're learning!

  • A Text Editor: our favourites are Atom, Brackets, and Sublime Text.
  • GitHub, which helps you organize your code and collaborate with others. Learn more in our Git and GitHub tutorial.

Languages

Here's a table of some of the most popular languages, though you certainly don't need to know all of them! Instead, pick 1 or 2 that fit what you're looking to make, and start from there. Plus, check out our sections in the sidebar for explanations and resources for each language!

Tip

One very common path is to learn Python, an object-oriented language, HTML, CSS, and Javascript. This is by no means the "correct" way to learn programming, but it covers a diverse set of languages, and can help you make almost anything!

Language Description Difficulty Used In
HTML and CSS Hyper Text Markup Language is a markup language that is used to make content on websites, while Cascading StyleSheets is used to make that content pretty! HTML and CSS are the barebones of any website. Easy Website Design, Templating
Javascript Javascript is a scripting language primarily designed for web interactions (like fancy buttons), but has evolved to be used almost anywhere! Many flavours of Javascript build on its core features, making it very versatile. Medium Website Design, Web Applications
Python Python is a dynamic scripting language with very readable, English-like syntax, but also packs a punch with many features and libraries. Python is a great language to start with! Medium Scripting, Modelling, Machine Learning
Ruby Ruby is an easily read, but powerful language with many uses in automation and web development. Ruby is often used with Rails to make web applications. Medium Scripting, Web Applications
Java Java is a compiled, object-oriented language based off of C. Java code only needs to be written once to be used everywhere, through the Java Virtual Machine. Recently, it is also used in Android app development. Hard Client-Server Applications, Android Applications
C C is a low-level, old, but still useful programming language that is the proverbial "grandfather" to most modern languages. It is still used today on low-level devices and programs. Hard Microcontrollers, Robotics, Operating Systems
C++ C++ is an object-oriented variation of C designed to improve upon C. C++ works on most Operating Systems, making it a popular choice in application development. Hard Desktop Applications, Servers, Hardware
C# C# is an object-oriented variation of C developed by Microsoft and designed to extend C's capabilities. As a result, it is often used with Windows products. Hard Windows Applications, Games

HTML and CSS

HTML (Hyper Text Markup Language) and CSS (Cascading StyleSheets) are the core foundation to any website on the internet. HTML is used to structure and write content, such as this paragraph; CSS is then used to make this paragraph look good, through colours, fonts, spacing, and many others. While HTML and CSS aren't "real" programming languages, they're a good start into making awesome things with computers, and are a super useful skill to have.

Recommended Tutorials

  • Marksheet is an online book, and also a staff favourite: it covers a broad range of topics thoroughly, but simply. Plus, it looks cool!
  • Codecademy is an interactive, online coding tutorial that gives you a hands-on experience on making your own website. And, it supports more than just HTML and CSS!
  • W3 Schools is a well-known website that has tutorials for learning HTML and CSS, as well as extensive documentation for everyting web related!
  • Web Design in 4 Minutes is a short tutorial that teaches you how to make a sleek website, in less than 4 minutes.

Further Learning

  • Web Design 101 is an eBook by WebFlow that goes in-depth on principles in web design, and how you can use them to your advantage!
  • SASS is a CSS preprocessor: it lets you use variables, on-build importing, mixins, and more to make your CSS super awesome!
  • Flexbox Froggy is a fun CSS-based game that teaches you how to use Flexbox, a new but amazing feature of CSS.

Useful Resources

  • Bootstrap is a super popular HTML, CSS, and JS framework that gives you a template to making awesome websites. We even use it on SSS!
  • Bulma is a more liteweight alternative to Bootstrap, that focuses on HTML and CSS. It's also a great kickstarter to a website!
  • GitHub Pages is a service provided by GitHub that provides you with a free domain and hosting, so the world can see your website for free!
  • Jekyll is a static-site generator written in Ruby that lets you "compile" your websites. We also use it on SSS!
