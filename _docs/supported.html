Notice

Student Server Space is still in alpha, so our supported project list is relatively small. We'll update this page as we support more types of projects. Stay tuned!

Status Table

Here's a quick and succint table that lets you know what we currently support on SSS. If your project type isn't here, then assume that we're not supporting it; that being said, let us know (we can try to get it working for you)!

  • Fully Supported
  • Partially Supported/Support Coming Soon
  • Not Supported
Project Type Type Description Special Requirements Supported
HTML Static Project HTML, CSS, and client-side JS project. No server side code. None
PHP-based Project PHP, HTML, CSS, and client-side JS project. PHP is run server side. None
Node.js Project Node.js centered project, can use HTML, CSS, JS assets. Node.js instance is run server side. npm install and npm start scripts with port parameters

Icebox

The Icebox is our list of project types that we aren't looking to support in the near future, for a variety of reasons. This doesn't mean we'll never support them, but don't expect us to allow these project types in the near future.

  • Ruby on Rails Web Applications
