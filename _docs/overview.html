Introduction

Hey there! You're probably reading this documentation site because you want to do something related with the Student Server Space, and that sounds awesome! Here, we have pages that tell you how to submit a project, deploy a project, manage the server, and everything in between. Plus, you'll find a set of policies and guidelines that make clear what's allowed on the Student Server Space, and what isn't. Let's take a look!
